# Hyacine.music Mobile

[简体中文](README.zh-CN.md) · [日本語](README.ja-JP.md)

Hyacine.music is a React Native and Expo music client for the Hyacine Server API.

## Features

- Background playback, queue management, media controls, and a full player
- Netease Cloud Music QR sign-in, recommendations, and personal playlists
- English, Simplified Chinese, and Japanese interfaces
- Native, liquid, and MIUIX visual styles with local appearance preferences

## Run

```bash
pnpm install
pnpm start
```

Use a Development Build to run the native client. Configure a reachable Hyacine Server address during first-run setup.

## Server

The backend, deployment instructions, and music-source integration details live in [hyacine-server](https://github.com/Ruoxi-TH/hyacine-server).
